## Windows? não!

Na maior parte do tempo, estou estudando e codando Java no Mac, pois estava tendo dificuldades de compatibilidade do XWayland com o IntelliJ no Arch. Até cogitei voltar para o Windows, mas hoje, com paciência, consegui encontrar em um fórum uma solução. Agora estou conseguindo usar normalmente e, como sou muito gente boa, vou deixar aqui a solução.

```bash
-Dawt.toolkit.name=WLToolkit
```

Adicione isso no Edit VM Options do IntelliJ e reinicie o IDE. Tudo deve funcionar.
